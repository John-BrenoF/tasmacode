# ECTE - Editor de Código via Terminal Edoz

Editor TUI simples com atalhos fáceis (Ctrl/Alt).  
Inspirado no Vim, mas para iniciantes.

## Atalhos
| Atalho         | Função                          |
|----------------|---------------------------------|
| `Ctrl + S`     | Salvar                          |
| `Ctrl + N`     | Novo arquivo (em branco)        |
| `Ctrl + F`     | Alternar barra lateral          |
| `Shift + A`    | Criar novo arquivo na pasta     |
| `R`            | Renomear item selecionado       |
| `D`            | Deletar item selecionado        |
| `Alt + P`      | Criar pasta                     |
| `Backspace`    | Voltar para pasta anterior      |
| `Alt + ←`      | Voltar no histórico de pastas   |
| `Alt + →`      | Avançar no histórico de pastas  |
| `Enter`        | Abrir item / Executar comando   |
| `Ctrl + E`     | Executar código no console      |
| `Alt + T`      | Alternar Console (terminal)     |
| `Ctrl + Q`     | Sair                            |
